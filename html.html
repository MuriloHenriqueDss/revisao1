O que é HTML?

Introdução ao HTML

HTML significa HyperText Markup Language. É a linguagem padrão para criar páginas web.

Tags HTML Básicas

Parágrafos

A tag < p >é usada para definir um parágrafo em HTML. Ela cria um bloco de texto com espaçamento superior e inferior, o que ajuda a separar o conteúdo de forma visualmente clara.

Listas

Listas Não Ordenadas
Tag: < ul > e < li >
A tag < ul > (unordered list) é usada para criar uma lista não ordenada, onde os itens são marcados com marcadores (geralmente bolinhas). Cada item da lista é definido com a tag < li > (list item).
Listas Ordenadas Tag: < ol > e < l >
A tag < ol > (ordered list) é usada para criar uma lista ordenada, onde os itens são numerados automaticamente. Assim como nas listas não ordenadas, os itens são definidos com a tag < li >.

Lista não ordenada

  • Item 1
  • Item 2
  • Item 3

Lista ordenada

  1. Item 1
  2. Item 2
  3. Item 3

Tabelas

As tabelas são usadas para organizar dados em linhas e colunas e são definidas pela tag < table >. A estrutura de uma tabela inclui:

Tag < table >: Define o início e o fim da tabela.

Tag < tr > (Table Row): Define uma linha dentro da tabela.

Tag < th > (Table Header): Define uma célula de cabeçalho, geralmente usada para rotular colunas e exibida em negrito e centralizada.

Tag < td > (Table Data): Define uma célula de dados que contém o conteúdo da tabela.

Imagem

A tag < img > é utilizada para inserir uma imagem em seu site, tendo também a < alt > funciona como uma "descrição".
