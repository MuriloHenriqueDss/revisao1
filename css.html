O que é CSS?

Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem usada para definir o estilo e a apresentação de documentos HTML. Ela permite que você controle a aparência de elementos em uma página web, separando o conteúdo (HTML) da apresentação (CSS). Com CSS, você pode definir propriedades como cores, fontes, espaçamentos e layout dos elementos em uma página web.

Conceitos Básicos

Seletores

O que são?: São padrões usados para selecionar os elementos HTML que você deseja estilizar. Exemplos: Seletores de tipo (p), classe (.classe), ID (#id), atributos ([type="text"]), e mais.

Propriedades e Valores

Propriedade: É o aspecto do estilo que você deseja alterar (por exemplo, color, font-size, margin). Valor: É o valor atribuído à propriedade (por exemplo, red, 16px, 10px). Sintaxe: A propriedade e o valor são separados por dois pontos, e cada declaração é terminada com um ponto e vírgula. Exemplo: color: red;.

Classes e IDs

Classes e IDs são usados para identificar elementos específicos e aplicar estilos a eles.

Cascata e Especificidade

O que é?: O processo de como as regras CSS são aplicadas quando existem várias regras para o mesmo elemento. Ordem de Aplicação: Regras são aplicadas com base na ordem em que aparecem no CSS, com regras mais específicas sobrepondo regras mais gerais.
